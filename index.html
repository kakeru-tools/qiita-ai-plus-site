Qiita-AI+

Qiita 執筆を AI が補助する Chrome 拡張

Chrome ウェブストアで入手

主な機能

📝 タイトル候補生成

記事内容から SEO に強いタイトル候補を AI が複数提案。

📊 LGTM 予測

過去の高 LGTM 記事と類似度を比較してスコア表示。

🏷 タグ最適化

発見されやすいタグを Qiita の統計から提案。

📈 いいね傾向分析

自記事のいいね時刻分布を可視化。

🤖 マルチ AI 対応

Claude / GPT / Gemini から選択可能。API キーはユーザー保管。

🔒 プライバシー

記事内容はユーザーが選んだ AI プロバイダーにのみ送信。Qiita-AI+ サーバーには送られない。

料金

Free: 基本生成 (月 30 回まで) は無料で使えます。
Pro (¥980/月): 無制限生成 + プロンプトテンプレライブラリ + 履歴管理 + マルチ AI 並列 + 製品特有機能。準備中。

使い方

  1. Chrome ウェブストアからインストール
  2. Qiita の新規記事ページを開く
  3. ツールバーに表示される 🤖 アイコンをクリック
  4. AI プロバイダー (Claude / GPT / Gemini) と API キーを設定
  5. 記事の下書き → AI 補助 → 公開

サポート

不具合報告 / 機能要望: GitHub Issues
