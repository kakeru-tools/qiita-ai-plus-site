Qiita-AI+ プライバシーポリシー

最終更新日: 2026-05-21

Qiita-AI+ (以下「本拡張機能」) は、お客様のプライバシーを最大限尊重します。本ポリシーは、本拡張機能が取り扱うデータの種類・目的・保管方法を説明します。

1. 本拡張機能が取得する情報

1.1 ローカル保存のみのデータ

本拡張機能は以下のデータを Chrome のローカルストレージ (chrome.storage.local) にのみ保存します。これらのデータは本拡張機能の運営者を含む第三者に送信されることはありません。

  • お客様が設定した AI プロバイダ (Claude または OpenAI)
  • お客様が選択した AI モデル
  • お客様が入力した API キー
  • 本拡張機能の表示設定

1.2 取得しない情報

本拡張機能は以下の情報を 一切取得しません。

  • 個人を特定する情報 (氏名・メールアドレス・住所等)
  • お客様の qiita.com アカウント情報
  • お客様の閲覧履歴
  • お客様の本文や記事の内容を運営者サーバーに送信すること

2. 外部送信

本拡張機能が外部に送信するデータは、お客様が明示的に「タイトル候補生成」「冒頭文生成」「章立て提案」ボタンを押した場合に限り、以下の宛先に対してのみ送信されます。

  • Anthropic API (https://api.anthropic.com) — お客様が Claude を選択した場合
  • OpenAI API (https://api.openai.com) — お客様が OpenAI を選択した場合

送信されるのは:

  • お客様自身の API キー (認証ヘッダー)
  • 現在執筆中の記事本文 (要約) ・タイトル

送信先サービスの利用規約とプライバシーポリシーは各社のものが適用されます:

  • Anthropic: https://www.anthropic.com/legal/privacy
  • OpenAI: https://openai.com/policies/privacy-policy

3. データの削除

ポップアップ画面の「クリア」ボタンで保存された設定および API キーを即座に削除できます。本拡張機能をアンインストールすると、ローカルストレージのデータも削除されます。

4. Cookie・トラッキング

本拡張機能は Cookie やトラッキング技術を一切使用しません。アナリティクスツール (Google Analytics 等) も組み込まれていません。

5. 第三者への提供

お客様のデータを第三者に提供することはありません。本拡張機能の運営者がお客様のデータにアクセスする手段を持ちません。

6. 変更

本ポリシーを変更する場合は、本ページおよび拡張機能のアップデート時の「更新内容」欄でお知らせします。

7. 連絡先

ご質問・ご懸念は GitHub Issues またはストアレビューでご連絡ください。
